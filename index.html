INTRO

For more than 10 years I’ve been working as an teacher and interpreter. I was doing literary and technical translations and teaching English to kids and adults. First in my hometown Perm, then in Moscow and last couple years in Shanghai. I was having a great time in my comfort zone untill I met JavaScript. Since then I quit my job and left my shell.

SKILLS

About a year ago I made my first webpage. Couple weeks were enough to feel comfortable with HTML and CSS. The next two months I read HTML5 and CSS3 documentation, completed ‘Responsive Web Design’ course on FreeCodeCamp and HTMLacademy course. I read an enormous number of articles, watched numerous video-tutorials, visited several events and workshops organised by CoderBunker and CodeBar.

I learned to make adaptive and cross-browser web design, switched from Sublume to Visual Studio Code. In couple days I learned to use Bootstrap framework. Wow.js and Animate.css libraries helped me apply animation to the elements of the page. Using preprocessors LESS and SASS, as well as BEM methodology made my coding process way easier and faster.

I began learning JavaScript with 'Intro to JS: Drawing & Animation' course on KhanAcademy. I’ve learned to make animated UX / UI elements. I’ve finished FreeCodeCamp course 'JavaScript Algorithms and Data Structures’. After couple months of practice writing vanilla JS code I’ve strated using JQuery library. I’ve started working with Terminal and a distributed version-control Git system.

I’ve been always interested in design. I have an experience using Adobe Photoshop, Illustrator, LightRoom, Xd, Figma. This year I’ve enrolled in the Design specialty at an online university.
